## Инструкция по установке проекта
### склонировать репозиторий
```
git clone https://github.com/KardashHleb/lab3.git
```
### установить зависимости
```
npm install
```

### запустить проект в режиме разработки
```
npm run dev
```

### собрать проект
```
npm run build
```
### добавить проект на GitHub Pages
```
npm run deploy
```
